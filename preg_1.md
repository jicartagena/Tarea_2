## Page Rank

Este codigo se basa en dos bloques, en el primero se definen las variables, estructuras y funciones a usar, mientras que en el segundo se realiza el algoritmo en si.
Las variables radican en las listas de nodos y conexiones a usar, la cantidad maxima de iteraciones a trabajar (en este punto se recomienda usar una cantidad menor a 5 si se requiere un trabajo rapido, ya que el tiempo aumenta exponencialmente con cada aumento de iteraci√≥n.), el umbral de convergencia y el damping factor a usar.

Para la entrega de resultados, unicamente se debe correr el segundo bloque de codigo.
